# TERMUX Frazix shell
Beautify your Termux 
### Installation
```bash
apt install git -y
cd $HOME
git clone https://github.com/Frazix12/F-shell.git
cd F-shell
chmod +x setup.sh
./setup.sh
```
The termux shell style gets changed!!
# Usage
###### To restore you old bashrc
```bash
cd F-shell && chmod +x uninstall.sh && bash uninstall.sh
